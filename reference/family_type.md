# Extracts the type of family in a consistent way

Extracts the type of family in a consistent way

## Usage

``` r
family_type(object, ...)

# S3 method for class 'family'
family_type(object, ...)

# Default S3 method
family_type(object, ...)
```

## Arguments

- object:

  an R object. Currently
  [`family()`](https://rdrr.io/r/stats/family.html) objects and anything
  with a [`family()`](https://rdrr.io/r/stats/family.html) method.

- ...:

  arguments passed to other methods.
