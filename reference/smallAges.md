# Lead-210 age-depth measurements for Small Water

A dataset containing lead-210 based age depth measurements for the
SMALL1 core from Small Water.

## Format

A data frame with 12 rows and 7 variables.

## Source

Simpson, G.L. (Unpublished data).

## Details

The variables are as follows:

- `Depth`

- `Drymass`

- `Date`

- `Age`

- `Error`

- `SedAccRate`

- `SedPerCentChange`
